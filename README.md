Мой пэт проект нацеленный на развитие в направлении фуллстек
Это будет простой сайт birthday-reminder.ru, суть которого будет в том, чтобы записывать дни рождения друзей/знакомых и он будет напоминать тебе о них за несколько дней, чтобы ты не забыл купить подарок и еще за денек до, чтобы ты не забыл поздравить
К сожалению для меня это проблема всю мою жизнь XD

Основной целью является созлдать качественный сайт, используя освоенные мной технологии и узнать новые в разработке, версионировании и разворачивании сайтов в целом.
